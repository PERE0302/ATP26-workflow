# ATP26 Workflow Documentation

This repository contains documentation for a web development workflow created as part of the ATP26 assignment. The project demonstrates the use of Markdown, AsciiDoc, and HTML documentation generated using Pandoc.

## Project Structure

├── assets/
│ └── images/
├── source/
│ ├── markdown/
├── README.md
└── index.md

## Technologies and Tools

- Visual Studio Code
- Pandoc
- Git
- GitHub
- GitHub Pages

## Documentation Formats

- Markdown (source)
- AsciiDoc (converted)
- HTML (converted)

## Assignment Requirements

This project fulfills the following requirements:

- Multiple Markdown documents with advanced Markdown elements
- GitHub repository hosting
- GitHub Pages for publishing documentation
